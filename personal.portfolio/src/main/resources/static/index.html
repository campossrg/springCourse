Web service para mejora de solicitudes web

Este proyecto forma parte de la renovación del proceso de solicitud de tarjetas para el transporte público. El servicio web hace de intermediario entre los usuarios y la empresa gestora de las tarjetas contactless. Es un proceso de intercambio de estados por los cuales se va realizando la petición dependiendo del perfil del solicitante.

Integración tarjeta T-Mobilitat

T-Mobilitat es el proyecto de integración de la tarjeta contacless para unificar todos los medios de transporte de Cataluña. De esta manera se promueve el uso de un único medio de pago tipo monedero recargable en cualquier punto de venta. La gestión se realiza a través de servidores de aplicaciones en cada operador estructurados en modulos que administran cada parte del sistema: Topología de red de transporte, equipos de campo, titulos de transporte, etc.
